This input is type="email" and required, so it's marked with an exclamation mark over there on the right. Once you enter a valid email, that exclamation mark will be replaced by a green check mark. This is how all HTML5 form validation features (required, patterns and types) are handled in Pocket.

This input has the class "invalid", you can use this for server-side validation.

You can also get the green checkmark by manually adding the class "valid" to an input.
